# Zeroyo
CLI tool for web application generator.

## Requirements
+ Node.js v8 or newer

## Installation

**npm**

    $ npm install tristachentw/zeroyo -g

**yarn**

    $ yarn add tristachentw/zeroyo global
## Usage

    $ zeroyo
