# Zeroyo
CLI tool for web application generator.

## Prerequisites
+ Node.js v8 or newer

## Installation

**npm**

    $ npm install tristachentw/zeroyo -g

**yarn**

    $ yarn global add tristachentw/zeroyo

## Getting Started

    $ zeroyo
